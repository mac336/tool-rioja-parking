# 05 · Módulo Incidencias — ⚠️ RETIRADO

> **Este módulo ya no existe.** El flujo antiguo (el vecino abría incidencias con
> fotos, comentarios, estados y moderación) se **retiró por completo** — código y
> base de datos — en la migración `0013`.
>
> Las incidencias son ahora un **tipo de "mensaje"** que publica la gestión. Ver
> **[16 · Mensajes y Tablón](16-modulo-mensajes-y-tablon.md)**.
>
> Para que un vecino reporte un problema (avería, etc.) ahora existe el **buzón
> privado** → ver **[17 · Buzón privado](17-modulo-buzon-privado.md)**. La gestión
> puede "convertir" un reporte del buzón en una incidencia pública.

Se conserva este archivo solo como referencia histórica del índice.
